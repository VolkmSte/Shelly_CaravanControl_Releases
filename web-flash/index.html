CaravanControl ESP32-S3 Web Flasher

Browser-Flash fuer ESP32-S3 (Chrome/Edge Desktop mit Web Serial).

Hinweis: Dieser Installer schreibt firmware.bin auf Offset 0x10000 und ist damit als Update-Pfad fuer bereits initialisierte Geraete gedacht.

Schritte

  1. Kanal waehlen (main oder beta).
  2. Version waehlen.
  3. Install klicken, seriellen Port waehlen und bestaetigen.
  4. Geraet nach dem Flash neu starten.

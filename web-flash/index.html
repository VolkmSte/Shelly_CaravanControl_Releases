CaravanControl ESP32-S3 Web Flasher

USB-Flash direkt im Browser (Chrome/Edge Desktop mit Web Serial).

Hinweis: Dieser Installer schreibt firmware.bin auf Offset 0x10000. Das ist der empfohlene Update-Pfad fuer bereits initialisierte Geraete.

Beta Installer

Aktueller Tag im Manifest: v1.0.15-beta.3

Schritte

  1. ESP per USB verbinden.
  2. Install-Button klicken.
  3. Seriellen Port waehlen und bestaetigen.
  4. Nach dem Flash neu starten.
